# Узнай город — Распознавание архитектурных объектов Санкт-Петербурга

**One-shot распознавание исторических зданий по фото с мгновенной выдачей исторической справки**

---

## Описание

**«Узнай город»** — это мобильное приложение, позволяющее пользователям распознавать архитектурные объекты в историческом центре Санкт-Петербурга по фотографии и получать подробную историческую справку о них. Сервис работает **офлайн**, не требует геолокации и не сохраняет пользовательские фото, что обеспечивает конфиденциальность и удобство для туристов и местных жителей.

---

## Ключевые возможности

- **One-shot распознавание** более 1000 уникальных зданий центра Санкт-Петербурга
- **Офлайн-режим** на мобильных устройствах (Android 8.0+, iOS 12+)
- **Быстрый отклик**: ≤5 секунд на распознавание
- **Приватность**: фото не сохраняются и не передаются на сервер
- **Историческая справка** по каждому объекту (интеграция с краеведческими базами)
- Поддержка работы на устройствах с разной производительностью

---

## Технологии и архитектура

### ML-стек
- **Базовая модель**: OS2D (One-Shot Object Detection) с fine-tuning
- **Бейзлайн**: ResNet-50 + k-NN (для сравнения)
- **One-shot learning**: обучение на ~5 000 размеченных изображениях с аугментацией
- **Целевая метрика**: mAP ≥ 90% на тестовой выборке
